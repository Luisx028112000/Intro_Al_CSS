Abreviado en sus siglas en inglés, Cascading Style Sheets, que significa hojas de estilo en cascada, el CSS es una función que se agrega a HTML que proporciona tanto a los desarrolladores de sitios Web, así como a los usuarios, más control sobre cómo se muestran las páginas. Con CSS, los diseñadores y los usuarios pueden crear hojas de estilo que definen cómo aparecen los diferentes elementos, como los encabezados y los enlaces. Estas hojas de estilo se pueden aplicar a cualquier página Web, y nos permite optimizarla para mejorar su posicionamiento.

Sus funciones basicas:

1. Selectores

  • Tipo: Selecciona todos los elementos de un tipo específico. Ejemplo: h1 { color: blue; }
  • Clase: Selecciona todos los elementos con una clase específica. Ejemplo: .miClase { color: red; }
  • ID: Selecciona un elemento con un ID específico. Ejemplo: #miID { color: green; }

    • 2. Propiedades Básicas:

    • Cambia el color del texto. Ejemplo: color: blue;

    • Cambia el color de fondo del elemento. Ejemplo: background-color: yellow;

    • Cambia el tamaño de la fuente. Ejemplo: font-size: 16px;

    • margin: el espacio exterior alrededor de un elemento. Ejemplo: margin: 10px;

    • Define el espacio interior alrededor del contenido de un elemento. Ejemplo: padding: 10px;

    3. Box Model

    •Width y Height: Establece el ancho y la altura de un elemento. Ejemplo: widt100px; height: 200px;

    •Border: Establece el borde del elemento. Ejemplo: border: 1px solid black;

    •Margin: Controla el espacio exterior de un elemento.

    •Padding: Controla el espacio interior de un elemento.

    4.Layout y Posicionamiento

  • Display: Define cómo se debe mostrar un elemento. Ejemplo: display: block;
  • Position: Define el tipo de posicionamiento de un elemento. Ejemplo: position: absolute;
  • Flexbox: Un sistema de diseño para alinear elementos en filas o columnas.

    • 5. Tipografía

    •Font-family: Establece la familia de fuentes para el texto. Ejemplo: font-family: Arial, sans-serif;

    •Font-size: Cambia el tamaño de la fuente. Ejemplo: font-size: 16px;

    •Font-weight: Define el grosor de la fuente. Ejemplo: font-weight: bold;

    6. Pseudo-clases y Pseudo-elementos

    •Pseudo-clases: Aplican estilos a un estado específico de un elemento. Ejemplo: a:hover { color: red; }

    •Pseudo-elementos: Aplican estilos a una parte específica de un elemento. Ejemplo: p::first-line { font-weight: bold; }

    • Los medios CSS

    Son reglas que permiten aplicar estilos CSS específicos en función de las características del dispositivo o del entorno donde se visualiza la página web, como el ancho de la pantalla, la orientación, la resolución, entre otros.

    Sintaxis Básica

    Ejemplos Comunes:

  • Estilo para pantallas pequeñas (como móviles)
  • Aplica el estilo cuando el ancho de la pantalla es de 600 píxeles o menos.


    • Estilo para pantallas grandes (como monitores):

  • Aplica el estilo cuando el ancho de la pantalla es de 1024 píxeles o más.
  • 3.Estilo para la orientación del dispositivo (vertical u horizontal)
  • Aplica el estilo cuando el dispositivo está en modo horizontal.

    • ¿Para Qué Sirven?

    Los medios CSS permiten crear diseños web responsivos, es decir, que se adaptan a diferentes tamaños de pantalla y dispositivos, ofreciendo una mejor experiencia de usuario en móviles, tablets, y computadoras de escritorio.
